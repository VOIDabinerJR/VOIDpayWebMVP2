Documentação do Botão de Pagamentos VOIDpay

Descrição

O botão de pagamentos Void Pay permite que os usuários realizem pagamentos de forma rápida e segura. Ele pode ser facilmente integrado em seu site usando HTML e CSS.

Integração

Para integrar o botão Void Pay ao seu site, adicione o seguinte código HTML:

<a href="https://voidpay.com/pay" class="void-pay-button">Pagar com VOIDpay</a>

Exemplo Completo

Veja abaixo um exemplo completo de como usar o botão de pagamentos VOIDpay:

<!DOCTYPE html>
            <html lang="pt-br">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>Exemplo de Botão Void Pay</title>
                <style>
                    .void-pay-button {
                        display: inline-block;
                        background-color: #0070ba;
                        color: white;
                        font-size: 16px;
                        padding: 10px 20px;
                        border: none;
                        border-radius: 5px;
                        cursor: pointer;
                        text-align: center;
                        text-decoration: none;
                        transition: background-color 0.3s;
                    }
                    .void-pay-button:hover {
                        background-color: #005fa3;
                    }
                </style>
            </head>
            <body>
                <a href="https://voidpay.com/pay" class="void-pay-button">Pagar com Void Pay</a>
            </body>
            </html>
